# Figma AI Export

This ZIP was generated by Figma to Code AI Export.

## Contents

- `index.html` - generated HTML/CSS from the selected Figma layer.
- `ai_prompt.txt` - prompt to paste into an AI coding tool.
- `images/` - local image assets used by the HTML.
- `references/` - screenshot(s) of the selected Figma layer or frame.

## How to use

1. Upload this ZIP to your AI coding tool.
2. Upload or reference the screenshot in `references/`.
3. Paste the contents of `ai_prompt.txt`.
4. Ask the AI to refine the generated HTML into production-ready code.
